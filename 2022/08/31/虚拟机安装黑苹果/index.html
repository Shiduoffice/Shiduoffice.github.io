虚拟机安装黑苹果

发表于2022-08-31|更新于2022-08-31
|阅读量:

准备工作

首先先在网上下载一个 VM16，这个是需要激活码的，自行百度就行。

下载 MacOS 的 ISO 镜像：本站链接

下载工具包：123 云盘

打开工具包中的 Unlocker 文件夹，右键管理员运行 win-install.cmd (运行前关闭 VMware 进程)

命令行出现 Finish 后按任意键退出

创建虚拟机

选择自定义

硬件兼容性默认就好

选择 MacOS 系统镜像

选择 MacOS 系统版本

给你的虚拟机取一个名字然后选择虚拟机要储存的位置，硬盘容量小的建议不要放在 C 盘

处理器根据自己电脑配置选择(至少选择 4 核)

内存根据自己电脑配置选择(至少 4096 GB)

网络类型选择”使用网络地址转换”

控制器类型选择”LSI Logic”

硬盘类型选择”SATA”

创建新虚拟磁盘

磁盘容量根据自己情况设置，选择”将虚拟磁盘储存为单个文件”

接下来一直点下一步就行

打开虚拟机设置，点击处理器选择虚拟化 Intel VT-x/EPT 或 AMD-V/RVI

打开虚拟机目录编辑.vmx 文件，在末尾添加以下代码：

smc.version = “0”

cpuid.0.eax = “0000:0000:0000:0000:0000:0000:0000:1011”

cpuid.0.ebx = “0111:0101:0110:1110:0110:0101:0100:0111”

cpuid.0.ecx = “0110:1100:0110:0101:0111:0100:0110:1110”

cpuid.0.edx = “0100:1001:0110:0101:0110:1110:0110:1001”

cpuid.1.eax = “0000:0000:0000:0001:0000:0110:0111:0001”

cpuid.1.ebx = “0000:0010:0000:0001:0000:1000:0000:0000”

cpuid.1.ecx = “1000:0010:1001:1000:0010:0010:0000:0011”

cpuid.1.edx = “0000:0111:1000:1011:1111:1011:1111:1111”

smbios.reflectHost = “TRUE”

hw.model = “MacBookPro16,1”

board-id = “Mac-E1008331FDC96864”

保存即可

安装 MacOS

打开虚拟机等待出现这个页面选择简体中文

双击”磁盘工具”点击第一个磁盘，点击抹掉

随意命名，下面的格式和方案不要动

退出磁盘工具，双击”Install macOS Ventura beta”

点击 Continue(可能因为是测试版所以界面变成了英文)

点击 Agree

选择要安装 macOS 的磁盘，再点击 Continue

等待一会儿

激活系统大家都会吧！这里我就不多说了(由于 CPU 问题我不能正常安装 macOS Ventura 所以下面我用 macOS Monterey 演示)

配置 macOS

激活后关机系统，接着打开虚拟机设置将”CD/DVD”中的镜像文件更改为工具包中”VMware Tools”文件夹下的”darwin.iso”映像文件

打开虚拟机，双击桌面上的”VMware Tools”再双击”安装 VMware Tools”

点击”继续”

点击”安装”

输入密码再点击”安装软件”

点击”好”

点击”打开安全性偏好设置”

点击左下角的 🔒 输入密码再点击”允许”

点击”重新启动”

点击”取消”

点击”重新启动”

拖入工具包中”优化”文件夹下的”Beamoff.zip”如果出现”辅助功能访问（事件）”请点击”打开系统偏好设置”

点击左下角的 🔒 输入密码再选择”vmware-tools-daemon”选项

再次拖入”Beamoff.zip”文件

点击”前往”-“电脑”

进入用户文件夹并解压”Beamoff.zip”文件

打开”系统偏好设置”并点击”用户与群组”

点击”登录项”并添加”Beamoff”文件

赞助

欢迎大家赞助我，赞助的越多我更新的越快
